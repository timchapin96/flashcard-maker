● ［グルメ］京麩・京ゆばの半兵衛
# S-ID:KN205_Gourmet_1-1-2-01 KNP:2008/02/25 SCORE:-62.74857 MOD:2009/06/30
● 川端五条の交差点に程近い場所に、京麩・京ゆばの老舗半兵衛があります。
# S-ID:KN205_Gourmet_1-1-3-01 KNP:2008/02/25 SCORE:-294.47384 MOD:2009/06/30
● 半兵衛では京麩、京ゆばの販売ももちろんしていますが、注目したいのが店舗の奥に構えられた瀟洒なスペースで供される“むし養い料理”。
# S-ID:KN205_Gourmet_1-1-4-01 KNP:2008/02/25 SCORE:-295.88624 MOD:2009/06/30
● 全て京麩と京ゆばを使った料理で献立が構成されている懐石料理を味あわせてくれる、ひと味違ったお店です。
# S-ID:KN205_Gourmet_1-1-5-01 KNP:2008/02/25 SCORE:-95.55834 MOD:2009/01/23
● 町屋を使った落ち着いた構えの店舗に入ると、着物の御婦人が奥の間へと案内して下さいます。
# S-ID:KN205_Gourmet_1-1-6-01 KNP:2008/02/25 SCORE:-334.69618 MOD:2009/06/30
● おくどさんに迎えられ履物を脱ぎ、ひとつひとつ念入りに選ばれたであろうと思わせる調度品で上品に飾られた部屋に腰を落ち着けその空間を楽しんでいると、とりどりに料理された京麩、京ゆばが次々と目の前に現れます。
# S-ID:KN205_Gourmet_1-1-7-01 KNP:2008/02/25 SCORE:-128.80914 MOD:2009/01/23
● その多彩さに、ひとつの素材にこんなにも調理法があるのか、と関心しきり。
# S-ID:KN205_Gourmet_1-1-8-01 KNP:2008/02/25 SCORE:-146.59589 MOD:2009/01/23
● 様々な表情を見せる京麩、京ゆばに舌鼓を打ち、いつの間にか心までゆったりとくつろいでいます。
# S-ID:KN205_Gourmet_1-1-9-01 KNP:2008/02/25 SCORE:-210.05945 MOD:2009/01/23
● お料理はもちろん、それを供する空間にまでこだわり抜かれた半兵衛では、最高の環境で京麩、京ゆばを味わうことができると言うことができるでしょう。
# S-ID:KN205_Gourmet_1-1-10-01 KNP:2008/02/25 SCORE:-187.87148 MOD:2009/06/30
● 事前に予約がいりますが、お値段は一人三千円と大変お手ごろで、誰でも京都の料理文化のひとつを気軽に楽しむことができます。
