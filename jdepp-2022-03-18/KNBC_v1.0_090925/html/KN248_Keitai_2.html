# S-ID:KN248_Keitai_2-1-2-01 KNP:2008/02/25 SCORE:-125.59233 MOD:2008/12/16
● 私はおしゃべりをするときは電話、連絡をするときはメールと決めている。
# S-ID:KN248_Keitai_2-1-3-01 KNP:2008/02/25 SCORE:-270.36998 MOD:2009/07/01
● 特に連絡をするときは、電話してしまうとついついダラダラしゃべってしまうこともあるし、相手が雑音のせいなどで聞き逃してしまったらもう一回言わないといけない。
# S-ID:KN248_Keitai_2-1-4-01 KNP:2008/02/25 SCORE:-142.11794 MOD:2008/09/22
● その点メールならば、相手が講義中だったとしても内容を確認できるし、内容を忘れてしまったとしても、いつでも読み返すことができるからだ。
