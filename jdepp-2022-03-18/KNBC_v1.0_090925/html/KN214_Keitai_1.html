● ［携帯電話］ポケベル打ちの極意
# S-ID:KN214_Keitai_1-1-2-01 KNP:2008/02/25 SCORE:-329.32656 MOD:2009/06/30
● 私の携帯電話には文字を打つときに普通のカナ入力に加えてポケベル入力が搭載されており、普段からそれを使っている。
# S-ID:KN214_Keitai_1-1-3-01 KNP:2008/02/25 SCORE:-266.77817 MOD:2009/06/30
● ポケベル入力とは、例えばあ＝１１、い＝１２、・・・、お＝１５、か＝２１、き＝２２、・・・というように各ひらがなに対して２桁の数字を対応させ、入力の際にはその２桁の数字を続けて打つ入力方式である。
# S-ID:KN214_Keitai_1-1-4-01 KNP:2008/02/25 SCORE:-238.30508 MOD:2009/06/30
● 単純に考えると、カナ入力では１文字あたり平均２．５回キーを押すのに対してポケベル入力なら２回ですんでしまう。
# S-ID:KN214_Keitai_1-1-5-01 KNP:2008/02/25 SCORE:-206.40484 MOD:2009/01/25
● また、同じ行の文字が連続したときに右キーを押す必要が全くないのも特徴である。
# S-ID:KN214_Keitai_1-1-6-01 KNP:2008/02/25 SCORE:-240.69723 MOD:2009/07/23
● 例えば、「青い海」と打つとき、カナ入力では［１右１１１１１右１１右１１１右７７］と打つのに対し、ポケベル入力では［１１１５１２１３７２］で済んでしまう。
# S-ID:KN214_Keitai_1-1-7-01 KNP:2008/02/25 SCORE:-394.80592 MOD:2009/06/30
● もちろんこれは極端な例だが、普段文字を打っているときに同じ行のカナが連続していらっとくる事はなくなるだろう。
# S-ID:KN214_Keitai_1-1-8-01 KNP:2008/02/25 SCORE:-316.04784 MOD:2009/01/25
● １打目に押すキーは仮名入力もポケベル入力も同じなので、私の場合は当初難なくポケベル打ちに移行する事が出来た。
# S-ID:KN214_Keitai_1-1-9-01 KNP:2008/02/25 SCORE:-211.24778 MOD:2008/11/15
● 携帯電話にポケベル入力機能が搭載されている方は、是非一度試してみてはいかがだろうか
