● ［京都観光］夏の下鴨神社
# S-ID:KN205_Kyoto_1-1-2-01 KNP:2008/02/25 SCORE:-216.14268 MOD:2009/06/30
● そもそも下鴨神社は水を司るお社ですが、とりわけ夏にはその特性を体感することができます。
# S-ID:KN205_Kyoto_1-1-3-01 KNP:2008/02/25 SCORE:-739.00794 MOD:2009/06/30
● 夏になると、境内にある御手洗川では足を川につけ涼むことができ、汗をかきかき参拝に来た人々はゆったりと水の恵みを楽しんでいます。
# S-ID:KN205_Kyoto_1-1-4-01 KNP:2008/02/25 SCORE:-200.38935 MOD:2009/06/30
● さらさらと流れる軽やかな水音は耳に心地よく、御手洗川の思いがけず冷たい水はほてった足をひんやりと癒してくれます。
# S-ID:KN205_Kyoto_1-1-5-01 KNP:2008/02/25 SCORE:-260.86553 MOD:2009/06/30
● 足を冷やしているだけでこんなにも涼しいなんて、とても気持ちいいね、とは御手洗川で涼んでいる人たちが口々に言う言葉。
# S-ID:KN205_Kyoto_1-1-6-01 KNP:2008/02/25 SCORE:-191.53888 MOD:2009/06/30
● 御手洗川だけでなく、下鴨神社が懐に抱える広大な糺の森に流れる小さな川のあちこちでも、川で涼む人々が見られます。
# S-ID:KN205_Kyoto_1-1-7-01 KNP:2008/02/25 SCORE:-427.42253 MOD:2009/07/22
● 特に土用の牛の日に執り行われる足つけ神事では、たくさんの人達がろうそくを手に御手洗川へ足を踏み入れ、神様へこの夏の健康を祈るとともに、夜の蒸し暑さを避けるように川の中で楽しげに涼を取ります。
# S-ID:KN205_Kyoto_1-1-8-01 KNP:2008/02/25 SCORE:-334.28534 MOD:2009/06/30
● 楽しげにはしゃぐ人々の喧騒の傍ら、人々が手にした無数のろうそくの火が夜の川面にゆらゆらと映る光景は何とも言えず神秘的です。
# S-ID:KN205_Kyoto_1-1-9-01 KNP:2008/02/25 SCORE:-185.35619 MOD:2009/06/30
● 暑さの激しいこの京都で、下鴨神社にお参りする人々は改めて水の恵みに思いをはせ、感謝の気持ちを抱くのではないでしょうか。
